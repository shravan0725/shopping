## 🛠️ Customization Guide

### Change Colors
Edit `css/style.css` line ~20:
```css
:root {
    --primary-color: #000000;      /* Change brand color */
    --accent-color: #d4af37;       /* Change accent */
    --text-color: #1a1a1a;         /* Change text */
}
```

### Add Products
Edit `js/cart.js` `addToCart()` function:
```javascript
const products = [
    { id: 1, name: 'Product', price: 99.99, ... },
    // Add more
];
```

### Change Font
Edit `index.html` line ~8:
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@300;700&display=swap">
```

### Modify Animations
Edit `css/style.css`:
```css
--transition: all 0.3s ease;  /* Adjust duration */
```

## 💡 Pro Tips

### Development
1. Use browser DevTools (F12) for debugging
2. Test on actual mobile devices
3. Use Lighthouse for performance audits
4. Check animations on slower devices
5. Validate HTML with W3C validator

### Customization
1. Make one change at a time
2. Test after each change
3. Use developer tools to inspect
4. Keep backups of original files
5. Document your changes

### Performance
1. Optimize images (use WebP)
2. Minify CSS and JavaScript
3. Enable gzip compression
4. Reduce font variants
5. Cache static assets
