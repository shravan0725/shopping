Accessibility Statement

LuxeShop is committed to ensuring digital accessibility for everyone, including people with disabilities. We are continuously improving our website's accessibility to meet or exceed WCAG 2.1 Level AA standards.

Our Accessibility Features

  • ✓ Keyboard navigation support throughout the site
  • ✓ Screen reader compatible with semantic HTML markup
  • ✓ High contrast mode and dark/light theme options
  • ✓ Adjustable text sizes and font options
  • ✓ Alternative text for all images and icons
  • ✓ Descriptive link text and ARIA labels
  • ✓ Captions and transcripts for video content
  • ✓ Reduced motion options for animations
  • ✓ Focus indicators on interactive elements
  • ✓ Mobile accessibility and responsive design

How to Navigate Our Site

Keyboard Navigation

You can navigate our entire website using the keyboard:

  • Tab: Move forward through interactive elements
  • Shift + Tab: Move backward through interactive elements
  • Enter: Activate buttons and links
  • Space: Toggle checkboxes and buttons
  • Arrow Keys: Navigate within dropdowns and sliders
  • Escape: Close modals and overlays

Screen Reader Support

Our site is fully compatible with popular screen readers including NVDA, JAWS, and VoiceOver. All content is properly labeled with ARIA attributes for context.

Text Sizing

Use your browser's zoom feature (Ctrl/Cmd + Plus) to increase text size. Our responsive design ensures readability at all zoom levels.

Color Contrast

All text meets WCAG AA standards for color contrast. Use our dark mode for additional options.

Accessibility Standards

We aim to comply with:

  • Web Content Accessibility Guidelines (WCAG) 2.1 Level AA
  • Americans with Disabilities Act (ADA) standards
  • Section 508 of the Rehabilitation Act
  • European Accessibility Act (EAA)

Known Issues

While we work continuously to improve accessibility, some third-party integrations may have limitations. Please contact us if you encounter any barriers.
